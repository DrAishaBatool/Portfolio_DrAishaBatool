Projects

  • Secure User-Centered Opinion-Sharing Platform

    A privacy-preserving, decentralized platform designed to allow users to securely share opinions and initiate review campaigns without revealing their identity. The system leverages Zero-Knowledge Proofs (ZKPs) and blockchain smart contracts to ensure data authenticity and user anonymity. Built for real-world deployment and pilot-tested with academic users.

    Technologies: Blockchain, ZK-SNARKs, Smart Contracts, Python, Web3, PostgreSQL
    Methods: Privacy-preserving cryptography, user authentication via ZKPs, incentivization via smart contracts, distributed ledger validation

  • Decentralized Blockchain-Based Peer Review System

    A transparent and tamper-proof peer review platform built on blockchain to eliminate bias, prevent manipulation, and maintain reviewer anonymity while ensuring accountability. It uses smart contracts for workflow automation and Zero-Knowledge Proofs to protect reviewer identities.

    Technologies: Ethereum Blockchain, Solidity, ZKPs, IPFS, Web3.js, Python
    Methods: Cryptographic hashing, immutable ledgers, public-key cryptography, smart contract-based review management, token-based reviewer incentive system

  • Traffic Sign Detection and Recognition System

    Developed a high-accuracy, lightweight computer vision model for real-time traffic sign detection and classification. The system integrates Convolutional Neural Networks (CNNs) with Extreme Learning Machines (ELMs) to balance performance and speed.

    Technologies: Python, MATLAB, TensorFlow, OpenCV
    Methods: Lightweight CNN architectures, Linear Discriminant Analysis (LDA), image preprocessing, feature extraction, model evaluation on benchmark datasets (CURE-TSR, TSDB, GTSRB)

  • Source Code Readability Prediction Tool

    A software analytics solution that predicts the readability of programming code using statistical modeling. It incorporates multiple regression analysis to study the impact of programming constructs on readability and includes a GUI-based readability estimation tool.

    Technologies: SPSS, .NET Framework, SQL Server
    Methods: Survey-based dataset collection, Multiple Linear Regression, readability metric formulation, prediction model development, statistical validation

  • Diabetes Assistance and Monitoring System

    Developed a desktop-based healthcare application to assist diabetic patients in managing their medical records and routines. It offers modules for patient profiles, doctor consultations, medication tracking, and lifestyle recommendations.

    Technologies: C#, MySQL, .NET Framework
    Methods: CRUD operations, user role-based access control, medical record digitization, disease management logic, patient–doctor interaction modules
